# AGENTS.md

## 前提

- このリポジトリは Django の簡易アプリです。

## 作業手順

1. 自分にアサインされたlssue を確認する
2. lssue の内容に応じたブランチを作成する
3. lssue に必要な変更を行う
4. 細かい単位でコミットする
5. ブランチをプッシュする
6. ブルリクエストを作成する

## Git ルール

- 上記の作業手順に従って作業してください。
- 'main'に直接コミットしないでください。
- コミットは細かい単位で行ってください。
- コミットメッセージは日本語で書いてください。

## ブルリクエスト

- ブルリクエストには日本語でコメントをつけてください。
